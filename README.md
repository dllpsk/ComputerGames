# 🎮 ComputerGames

**ComputerGames** — приложение для работы с каталогом компьютерных игр. Пользователь может загружать каталог игр, выбирать формат хранения данных, искать игры с помощью фильтрации и сортировки, просматривать подробную информацию об играх, а также управлять содержимым каталога.

## 📌 Основные возможности

- Загрузка каталога игр из выбранной папки.
- Поддержка форматов хранения данных **JSON** и **XML**.
- Фильтрация и сортировка игр по различным параметрам.
- Просмотр подробной информации об игре.
- Добавление новых игр в каталог.
- Удаление игр из каталога.
- Удобный графический интерфейс на WPF.

## 🖥️ Главное окно (ComputerGame)

Главное окно отображается при запуске приложения.

### Элементы интерфейса

- Надпись **«Добро пожаловать»**.
- Кнопка **«Перейти в каталог»** для открытия каталога игр.
- Выбор папки с данными игр.
- Выбор формата хранения данных:
  - JSON
  - XML

После выбора папки и формата пользователь может перейти к работе с каталогом игр.

## 🎲 Окно каталога игр (GameCatalog)

Основное окно для поиска и просмотра игр.

### Возможности

#### Фильтрация и сортировка

Пользователь может выполнять поиск игр с помощью фильтров:

- По платформе.
- По режиму игры.

#### Выбор игры

После применения фильтров отображается список подходящих игр.

**Кнопки:**

- **Выбрать игру** — выбор игры из списка.
- **Показать данные** — открывает окно с подробной информацией о выбранной игре.

#### Управление каталогом

Кнопка **«Открыть каталог»** позволяет перейти в окно управления каталогом игр.

## 📋 Окно информации об игре (GameInfo)

Отображает подробную информацию о выбранной игре.

### Доступные данные

- Изображение игры.
- Название.
- Жанр.
- Рейтинг.
- Возрастное ограничение.
- Дата выхода.
- Тип игры.
- Описание.

Данное окно предоставляет пользователю полную информацию об игре в удобном для просмотра виде.

## ⚙️ Окно управления каталогом игр

Предназначено для редактирования каталога.

### Возможности

- Просмотр полного списка игр.
- Добавление новой игры.
- Удаление существующей игры.
- Обновление каталога после внесения изменений.

## 💾 Поддерживаемые форматы данных

### JSON

Основной формат хранения информации об играх.

### XML

Альтернативный формат хранения данных.

Пользователь самостоятельно выбирает формат работы при запуске приложения.
